# base58

Generate random strings with Base58 alphabet.

## Installation

Add this to your application's `shard.yml`:

``` yaml
dependencies:
  email:
    github: valenciaj/crystal-base58
```

## Usage

``` crystal
require "base58"

# Generating a 32 chars string length
puts Base58.random 32
```
